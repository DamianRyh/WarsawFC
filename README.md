# WARSAW FC Admin Only 1.9 — Card Modes

Dodane w generatorze kart:
- typ karty: Ustawka / Sezon / Hybryda
- wybór konkretnej ustawki dla karty ustawki i hybrydy
- karta ustawki pokazuje dane z jednego eventu
- karta sezonu pokazuje dane całego sezonu
- karta hybrydowa pokazuje sekcję „Ta ustawka” + sekcję „Sezon”

Zachowane:
- eksport PNG
- style karty
- zdjęcie zawodnika
- WE ARE WARSAW FC
- brak miejsca na logo w górnym pasku karty
