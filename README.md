# WARSAW FC Admin Only 1.1 — Advanced Stats

Dodane statystyki:
- Team Goals AVG — gole zespołu na mecz przypisane każdemu zawodnikowi teamu.
- Team Points AVG — punkty zespołu na mecz.
- Impact Score — gole + punkty zespołu + bilans + obecność.
- Silent Impact — wpływ zespołowy bez premiowania samych goli.
- Rozszerzony ranking sezonu.
- Rozszerzony profil gracza.
- Zapis wyników `scores` w historii ustawki.
