# WARSAW FC Base 1.0 Clean

Stabilna, uporządkowana wersja aplikacji.

## Struktura

```txt
index.html
styles.css
config.js
supabaseClient.js
app.js
TEST_CHECKLIST.md
SUPABASE_NOTES.md
```

## Funkcje w tej wersji

- logowanie/rejestracja przez Supabase,
- Guest/Public,
- Admin/Public/TV,
- terminarz 1–3 boiska,
- 6–20 zespołów,
- strzelcy w meczu,
- tabela live,
- lista strzelców,
- profile graczy,
- historia ustawek,
- ranking sezonu,
- eksport/import local JSON,
- seed ustawek S03E03–S03E12.

## Deploy
