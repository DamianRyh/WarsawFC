# WARSAW FC Base 1.0 Clean — SINGLE FILE

Ta wersja ma wszystko w jednym pliku `index.html`:
- style,
- konfigurację,
- logikę aplikacji,
- Supabase client wrapper.

Użyj tej wersji, jeśli po wgraniu na GitHub strona wyglądała jak surowy HTML bez stylów.

## Deploy

Do GitHub wrzuć tylko:
- `index.html`
- `README.md`
- `vercel.json`
- `.gitignore`

Najważniejsze: `index.html` musi być w głównym katalogu repo.
