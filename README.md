# WARSAW FC Matchday OS 1.5

System do prowadzenia ustawek Warsaw FC: terminarz, live score, tabela, strzelcy, profile, ranking sezonu, generator kart, tryb TV i widok publiczny dla graczy.

## Najważniejsze zmiany w 1.5

- nowy widok **Szybki Admin** do obsługi ustawki z telefonu,
- nowy **Widok publiczny** bez edycji wyników i ustawień,
- cofanie ostatniego gola,
- autosave z komunikatem zapisu,
- mocniejsze potwierdzenia przy resetach i usuwaniu danych,
- backup JSON po eksporcie i po zapisaniu ustawki,
- przełącznik trybu wizualnego: **Warsaw FC** / **Partner mode**,
- uporządkowana wersja aplikacji: 1.5.0.

## Zachowane funkcje

- liczba zespołów: 2–10,
- domyślnie 6 zespołów,
- generator kart graczy,
- style Premium / Sport / Story / Post 4:5 / Minimal Poster,
- kalendarz sezonu,
- terminarz,
- ranking,
- profile,
- TV mode,
- eksport/import JSON.

## Ważna uwaga

To nadal statyczna aplikacja MVP. Panel admina działa po kodzie zapisanym w frontendzie, więc nie należy traktować tego jako pełnego zabezpieczenia produkcyjnego. Docelowo warto dodać backend, logowanie i wspólną bazę danych.
