# WARSAW FC Admin Only 1.0

Stabilna, uproszczona aplikacja tylko do własnego zarządzania.

## Brak rejestracji
Nie ma kont graczy, resetów haseł ani publicznego logowania.

## Dostęp
Domyślny kod admina: `WFC-ADMIN`

## Dane
Dane zapisują się lokalnie w przeglądarce. Używaj:
- Eksport — kopia zapasowa JSON
- Import — wczytanie kopii

## Funkcje
- terminarz 1–3 boiska,
- 6–20 zespołów,
- rewanże,
- timer,
- strzelcy w meczu,
- tabela live,
- profile graczy,
- historia ustawek,
- ranking sezonu,
- tryb TV,
- mobile.
